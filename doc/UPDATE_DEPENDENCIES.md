# 依存関係更新ガイド

## 更新方針

依存関係を段階的に更新し、各ステップでテストを実施します。

## 更新手順

### フェーズ1: コア依存関係の更新
- numpy (1.26.0 → 2.4.0) - メジャーアップデートに注意
- pandas (2.2.3 → 2.3.3)
- scipy (1.15.1 → 1.16.3)

### フェーズ2: 機械学習ライブラリの更新
- torch (2.2.2 → 2.9.1)
- sentence-transformers (3.3.1 → 5.2.0)
- scikit-learn (1.6.1 → 1.8.0)

### フェーズ3: 音声/動画処理ライブラリの更新
- av (11.0.0 → 16.0.1)
- opencv-python (4.7.0.72 → 4.12.0.88)
- mediapipe (0.10.20 → 0.10.31)
- scenedetect (0.6.5.2 → 0.6.7.1)

### フェーズ4: 話者分離ライブラリの更新
- pyannote.audio (3.3.2 → 4.0.3) - 互換性対応済み
- pyannote.core (5.0.0 → 6.0.1)

### フェーズ5: ユーティリティライブラリの更新
- matplotlib (3.10.0 → 3.10.8)
- nltk (3.9.1 → 3.9.2)
- psutil (6.1.1 → 7.1.3)
- pynvml (12.0.0 → 13.0.1)

## 注意事項

- numpy 2.xはメジャーアップデートのため、互換性に注意が必要
- torch 2.9.1は大幅な更新のため、慎重にテストが必要
- sentence-transformers 5.2.0はメジャーアップデートの可能性あり
